# Procioni in Cina — sito GitHub Pages

Sito statico multipagina per il viaggio in Cina del 9–22 agosto 2026.

## Pagine

- `index.html` — sintesi del viaggio
- `itinerario-dettagliato.html` — programma ora per ora, mappe e trasferimenti
- `cibi.html` — 85 specialità gastronomiche con fotografie remote illustrative
- `attrazioni.html` — schede culturali e fotografie remote illustrative
- `frasi.html` — 280 frasi italiano–cinese, ricerca, preferiti, copia, pronuncia e modalità cartello

## Pubblicazione

Caricare nella root del repository tutti i file e l'intera cartella `assets/`. Non caricare soltanto lo ZIP.

Le fotografie sono caricate da link esterni tramite LoremFlickr e vengono richieste soltanto quando il contenuto si avvicina allo schermo. Senza connessione rimangono visibili testi, dati, itinerario e frasario; le immagini, le mappe e Google Translate possono non essere disponibili.
